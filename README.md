### Dataset

Your file structure will be like:

---

```
├── [Your Dataset Path]
    ├── JPEGImages
        ├── img_001.jpg
        ├── img_002.jpg
        └── ...
    ├── SegmentationClass
        ├── label_001.png
        ├── label_002.png
        └── ...
    └── ImageSets
    	└──Segmentation
    		└──split.txt
```
Modify your dataset path in train.py
``
Dataset_path  = 'YourDataset'
`` 
.

### Pretrained Backbone
